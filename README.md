# si1v3rApi

### 前端

前端项目我本地使用的是yarn来管理包依赖

#### 安装依赖

```
yarn
```

#### 本地运行项目

本地启动项目如果不需要mock的虚拟数据，使用命令

```
yarn start
```

至此运行前端在地址http://localhost:8000







### 后端

后端项目运行中端口http://localhost:8001
后端接口管理文档地址为http://localhost:8001/api/doc.html
